# 用户管理

管理后台的用户管理功能。

## 查看用户列表

在管理后台点击"用户管理"标签，可以查看所有注册用户的信息。

## 用户信息

每个用户显示以下信息：

| 字段 | 说明 |
|------|------|
| **用户** | 头像 + 昵称 + 邮箱 |
| **角色** | `admin`（管理员）或 `user`（普通用户） |
| **邮箱验证** | 已验证 / 未验证 |
| **注册时间** | 用户注册时间（北京时间） |

## 用户角色

### 管理员（admin）

- 可以访问管理后台
- 可以删除任意评论
- 可以管理系统设置
- 可以管理域名白名单

### 普通用户（user）

- 可以发表评论
- 可以回复评论
- 可以点赞评论
- 可以删除自己的评论

## 邮箱验证

用户注册后需要通过邮箱验证：

1. 用户注册时发送 6 位 OTP 验证码
2. 用户输入验证码完成验证
3. 验证后邮箱状态变为"已验证"
4. 只有已验证的用户才能登录

## 用户头像

系统自动生成头像：
- 使用昵称首字母
- 浅蓝色渐变背景
- 圆形头像样式

## 权限管理

目前角色权限：
- 管理员权限在数据库中设置
- 通过 `role` 字段区分用户角色
- 登录时验证角色权限

## 查看用户评论

要查看特定用户的评论：

1. 记录用户昵称或邮箱
2. 在评论管理中使用搜索功能
3. 输入昵称或邮箱进行筛选
