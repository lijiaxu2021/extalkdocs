# 评论管理

管理后台的评论管理功能详解。

## 界面布局

评论管理界面采用左右分栏布局：

### 左侧主区域

**全部评论列表**
- 表格形式展示所有评论
- 显示：用户、内容、URL、时间、操作
- 支持关键词搜索
- 支持按 URL 筛选

### 右侧侧边栏

**🕒 最新评论**
- 显示最新 10 条评论
- 快速预览评论内容
- 快捷删除操作
- 点击 URL 快速筛选

**📄 文章 URL 导航**
- 卡片式展示所有页面 URL
- 显示每个页面的评论数量
- 悬停动画效果
- 点击即可筛选对应页面评论

## 筛选功能

### 按 URL 筛选

1. 使用顶部下拉选择框
2. 选择特定页面 URL
3. 或直接点击 URL 卡片

### 关键词搜索

支持搜索：
- 用户昵称
- 评论内容
- 页面 URL

## 删除评论

1. 找到要删除的评论
2. 点击"删除"按钮
3. 确认删除操作

⚠️ **注意**: 删除操作不可恢复，请谨慎操作。

## 统计信息

顶部显示三个统计卡片：

- **总评论数** - 所有评论的总数
- **页面数** - 有多少个不同的页面 URL
- **根评论数** - 一级评论（不含回复）的数量

## 交互特性

- **快速筛选** - 点击任意 URL 即可筛选
- **实时更新** - 删除评论后立即刷新列表
- **响应式设计** - 移动端自动调整布局
- **悬停效果** - 卡片和按钮都有悬停动画
