# 管理后台

ExTalk 提供现代化的管理后台，方便管理员管理评论、用户和系统设置。

## 访问管理后台

访问地址：`https://your-worker.workers.dev/upxuuadmin`

首次访问需要管理员登录。

## 登录

使用管理员邮箱和密码登录：

- **邮箱**: 配置的环境变量 `ADMIN_EMAIL`
- **密码**: 配置的环境变量 `ADMIN_PASS`

登录需要完成 hCaptcha 人机验证。

## 功能模块

### 1. 评论管理

- **全部评论列表** - 显示所有评论，支持搜索和筛选
- **最新评论** - 侧边栏显示最新 10 条评论
- **URL 导航** - 按页面 URL 分组显示评论数量
- **快速筛选** - 点击 URL 即可筛选该页面的评论
- **删除评论** - 支持单条评论删除

### 2. 域名管理

配置允许调用 API 的域名白名单：

- 添加允许的域名
- 支持通配符（如 `*.example.com`）
- 删除已授权的域名

### 3. 用户管理

查看所有注册用户：

- 用户 ID、昵称、邮箱
- 用户角色（普通用户/管理员）
- 邮箱验证状态
- 注册时间

### 4. 系统设置

- **IP 属地精度** - 省份级别或城市级别
- **评论长度限制** - 单条评论最大字符数
- **邮件同步间隔** - 新评论汇总邮件的发送频率

## UI 特点

- **浅蓝色主题** - 现代化的视觉设计
- **响应式布局** - 适配各种屏幕尺寸
- **左右分栏** - 主内容区 + 侧边栏
- **卡片式设计** - 清晰的信息层级
